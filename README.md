## Install

- Exit Xcode
- Run `./install.sh`
- Open Xcode and select `Github (Dark)` or `Github (Light)` theme

Enjoy it.

Suggested mono fonts: [Intel One Mono](https://github.com/intel/intel-one-mono), [Cascadia Code](https://github.com/microsoft/cascadia-code)
